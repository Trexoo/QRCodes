Quick Start

Generate a QR code using cURL:

curl -X GET "https://api.qrcore.com/v1/generate" \ -H "Authorization: Bearer YOUR_API_KEY" \ -G -d "data=https://example.com" \ -d "type=url" \ -d "format=png" \ -d "size=300"

Using JavaScript:

const response = await fetch('https://api.qrcore.com/v1/generate', { headers: { 'Authorization': 'Bearer YOUR_API_KEY' }, params: { data: 'https://example.com', type: 'url', format: 'png', size: 300 } }); const qrImage = await response.blob();
